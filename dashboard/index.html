Steam moves are sharp, sustained consensus odds shifts that front-run the news. Steamline detects them in TxLINE's demargined stable line and trades every signal twice: follow rides the move, fade bets it is an overreaction. Positions and settlement run on an open Anchor program on Solana devnet where any strategy can register a book and compete. Production thresholds are 3pp / 2% edge; the defaults here are calibrated for these quiet pre-match windows. Replays are deterministic: same data, same settings, same result, every time.
